# Console Space Simulation (C)
A console-based space simulation project written in C that simulates
basic physics experiments across planets in the Solar System.

## Project Purpose
- Improve algorithmic thinking and problem solving skills
- Practice modular programming with functions and pointers
- Apply physics formulas in a software environment
- Build a text-based interactive console application

## Implemented Experiments
- Free Fall
- Upward Throw
- Weight Calculation
- Gravitational Potential Energy
- Hydrostatic Pressure
- Archimedes' Buoyant Force
- Simple Pendulum Period
- Rope Tension
- Elevator Motion

## How to Compile and Run
```bash
gcc src/main.c -o simulation
./simulation
```

## Technologies
- C Programming Language
- GCC Compiler
- Command Line Interface
